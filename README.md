# Proyecto Residencias UNAL - Estructuras de Datos

## Descripción
Sistema de asignación de cupos para residencias universitarias basado en puntaje socioeconómico. El sistema gestiona el registro de estudiantes y asigna cupos automáticamente priorizando a los estudiantes con menor puntaje (mayor vulnerabilidad).

## Estructura del Proyecto

    * /ui: Interfaz gráfica (Ventanas y formularios)
